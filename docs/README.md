## ドキュメント管理原則

1. **一貫性** - すべてのドキュメントは共通のフォーマットと用語を使用
2. **最新性** - コード変更に伴いドキュメントも同時更新
3. **トレーサビリティ** - 要件から実装までの追跡が可能
4. **自己完結性** - 各ドキュメントは必要な文脈を含み単独でも理解可能
